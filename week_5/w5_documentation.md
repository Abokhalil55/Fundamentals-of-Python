# Week 5 Documentation

## 1.1 Problem Statement

Create a simple Python program that calculates a customer's cafe bill and prints a receipt.

## 1.2 Inputs

- Customer name
- Coffee quantity
- Tea quantity
- Sandwich quantity

## 1.3 Outputs

- Customer name
- Quantity of each item
- Total bill in RM

## 1.4 Typical Process Flow

1. Ask for the customer's name.
2. Ask for the quantity of each item.
3. Multiply each quantity by its price.
4. Add all item costs to get the total.
5. Print the receipt.

## 1.5 Constraints

- Coffee costs RM 8.50.
- Tea costs RM 6.00.
- Sandwich costs RM 12.00.
- Quantities must be whole numbers and cannot be negative.

## 2. Problem Decomposition

1. Get the customer details and item quantities.
2. Calculate the total bill in a function.
3. Print the receipt in another function.
4. Run the functions from the main program.

## 3. Pseudocode

```text
START

INPUT customer name
INPUT coffee quantity
INPUT tea quantity
INPUT sandwich quantity

CALCULATE coffee cost
CALCULATE tea cost
CALCULATE sandwich cost
ADD all costs to get total

DISPLAY receipt
DISPLAY customer name
DISPLAY item quantities
DISPLAY total

END
```
