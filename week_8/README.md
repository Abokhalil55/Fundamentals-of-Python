# Food Ordering Application

## 4.1. Purpose of the Application
This application is designed to streamline a simple food ordering process for a small bakery or café specializing in cakes and muffins. It prompts users for their order details, automatically handles data input validation, calculates subtotals with a 5% service charge, applies delivery fees if applicable, and generates a clean, readable receipt for the customer.

## 4.2. Tech Stack
* **Language:** Python 3.x
* **Core Concepts Used:** Modular programming (multi-file implementation), loops and conditional validation, console I/O, string formatting.

## 4.3. How to Use
1. Ensure you have Python installed on your system.
2. Clone or download the project files (`app.py`, `customer.py`, `receipt.py`) into the same folder.
3. Open your terminal or command prompt, navigate to the folder, and run:
   python main.py
